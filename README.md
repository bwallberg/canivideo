# canivideo

A JavaScript library and web app for detecting video codec and DRM support in browsers.

## What is this?

**canivideo** helps you determine whether a browser can play specific video formats by checking:

- **Video codec support** (H.264, H.265, VP9, AV1, etc.)
- **DRM support** (Widevine, PlayReady, FairPlay, etc.)
- **Container format compatibility** (MP4, WebM, HLS, etc.)
